About Wine Recommender

Wine Recommender

Wine Recommender uses a machine learning model to recommend a wine variety to a user based on an input tweet with a description of a desired wine.

The components of the Wine Recommender are:

  • Twitter BOT that listens for a tweet requesting a wine recommendation, calls the machine learning model to get the recommendation, and then tweets two responses:
    1. Recommended variety of wine, along with the 3 top-rated wineries for that variety, along with their average price
    2. Recommended variety of wine, along with the 3 least expensive wineries for that variety, along with their average price
  • Machine learning model that reads a CSV file with Wine Enthusiast descriptions of wines, the wineries, points, and prices. The model uses a neural network and natural language processing to be able to predict the wine variety based on an input description.
  • Graphs and tabular data about the wines, varieties, and vineyards created using Tableau.


To learn more about this project, including the details of neural network, please visit the code on GitHub.
